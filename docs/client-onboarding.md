# Hype Up AI クライアント利用ガイド

作成日: 2026-05-11  
対象: Hype Up AI のクライアントユーザー（client_owner / client_member）

---

## 1. ログイン方法

Hype Up AI はパスワードレス（マジックリンク）方式でログインします。

1. ブラウザで `https://your-domain.com/login` を開く
2. 招待時に連絡を受けたメールアドレスを入力
3. 「ログインリンクを送信」をクリック
4. 受信したメールの「ログイン」リンクをクリック
5. 自動的にログインされてトップページへ遷移する

> **注意:** リンクの有効期限は 60 分です。期限切れの場合は再度ログインページから送信してください。

## 2. CSV フォーマット

### 必須列

| 列名 | 説明 |
|------|------|
| `body` | レビュー本文（必須） |

### 任意列

| 列名 | 説明 | 例 |
|------|------|----|
| `rating` | 星評価（1〜5 の整数） | `5` |
| `reviewer` | 投稿者名（匿名可） | `田中` |
| `reviewed_at` | 投稿日 YYYY-MM-DD 形式 | `2024-01-15` |
| `source` | 取得元サイト | `amazon`, `rakuten`, `google` |

### サンプル

```csv
body,rating,reviewer,reviewed_at,source
"とても使いやすく満足しています",5,田中,2024-01-15,amazon
"価格の割に品質が良い",4,鈴木,2024-02-03,rakuten
"少し重いが機能は十分",3,,2024-03-10,
```

### 注意事項

- 文字コードは **UTF-8** を推奨（Excel で保存する場合は「CSV UTF-8（コンマ区切り）」を選択）
- ヘッダー行は必須
- 最大 **5,000 件** まで処理可能
- `body` が空の行は自動的にスキップされます

## 3. 分析手順

### ステップ 1: CSV をアップロードする

1. トップページ（`/`）を開く
2. 「新規分析を開始」フォームにプロジェクト名を入力
3. 業界テンプレートを選択（美容・コスメ / 食品・飲料 / 英語スクール など）
4. CSV ファイルをドラッグ＆ドロップまたはファイル選択
5. 「アップロード」ボタンをクリック

### ステップ 2: AI 分析を実行する

1. アップロード完了後、プロジェクト詳細ページに自動遷移
2. 「AI 分析を開始」ボタンをクリック
3. 分析中は「分析中...」と表示（通常 1〜3 分）
4. 「完了」になったら結果を確認

---

## 4. レポート確認方法

分析完了後、プロジェクト詳細ページに以下が表示されます。

| セクション | 内容 |
|-----------|------|
| 評価ポイント | 高評価を集めている要素のトップランキング |
| 不満・改善点 | 低評価・クレームに多い要素のランキング |
| 購買理由 | 顧客が購入を決めた動機のパターン |
| 顧客タイプ | どんな属性・目的の顧客が多いか |
| 訴求ワード | 共感されやすいキーワード一覧 |
| LP 改善提案 | レビューをもとにした具体的なコピー案 |

---

## 5. PPTX 出力方法

1. プロジェクト詳細ページを開く
2. 右上の「PPTX ダウンロード」ボタンをクリック
3. `[プロジェクト名]_report.pptx` が自動ダウンロード

> **補足:** 1 枚サマリー版（One-Pager）は「1 枚サマリー表示」から確認・ダウンロードできます。

## 6. 競合比較の使い方

複数プロジェクトを横断比較して、差別化ポイントと共通課題を抽出します。

### 手順

1. ヘッダーの「競合比較」または `/compare` を開く
2. 比較したいプロジェクト（自社・競合）を 2〜4 件選択
3. 「比較レポートを生成」をクリック
4. 比較レポートページに遷移（通常 1〜2 分）
5. 「差別化」「台頭中」「共通」のカテゴリで整理された結果を確認

### 活用例

- 自社 EC と競合 EC の「不満点」を比較し、差別化施策を検討
- 複数商品の「購買理由」を比較し、訴求軸を統一する

## 7. よくあるエラー

### アップロード時

| エラー | 原因 | 対処法 |
|--------|------|--------|
| `body 列が見つかりません` | ヘッダーに `body` 列がない | CSV の列名を確認してください |
| `文字化けしています` | Shift-JIS など UTF-8 以外の文字コード | Excel で「CSV UTF-8（コンマ区切り）」で保存し直す |
| `5,000 件を超えています` | レビュー数が上限超過 | 件数を 5,000 件以下に絞る |
| `ファイルサイズが大きすぎます` | 10MB 超 | 件数を減らすか分割してアップロード |

### 分析時

| エラー | 原因 | 対処法 |
|--------|------|--------|
| `分析エラー` | AI API のタイムアウトなど | 「再分析」ボタンで再試行。繰り返す場合はサポートへ連絡 |
| `レビューが少なすぎます` | 有効なレビューが 10 件未満 | レビュー数を増やして再アップロード |

### ログイン時

| エラー | 原因 | 対処法 |
|--------|------|--------|
| `組織が見つかりません` | 招待が完了していない | 担当者（platform_admin）に組織への追加を依頼 |
| `メールが届かない` | 迷惑メールフォルダ / メールアドレス誤り | 迷惑メールを確認。正しいアドレスで再送信 |
| リンクが無効 | 60 分の有効期限切れ | ログインページから再度メール送信 |
